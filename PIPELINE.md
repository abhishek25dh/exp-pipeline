# Full Pipeline: Explainer Video Generator

## Overview

```
Raw paragraph text
    → Scene/Layout planning  (layout_selector.py)
    → Director script        (run_and_audit.py / layout_tester.py)
    → Image generation       (generate_prompts.py + image.py)
    → Timings                (assets/timings.py)
    → Final video render     (render_video.py)
```

---

## PHASE 0 — Prerequisites (manual, one-time setup)

These files must exist before the pipeline runs. They are NOT generated by this project.

| File | Description |
|------|-------------|
| `assets/para/para_N.json` | Raw paragraph text with key `"para_N"` |
| `assets/scenes_audio/scene_N.mp3` | Voiceover audio per scene |
| `assets/scenes_audio/scene_N_transcript_full.json` | Word-level transcript with timestamps (from Whisper or similar) |
| `sound_effects/click.mp3` | SFX files |
| `sound_effects/pop.mp3` | SFX files |
| `sound_effects/whoosh.mp3` | SFX files |
| `sound_effects/typing.mp3` | SFX files |

## PHASE 1 — Scene + Layout Planning

**Script:** `layout_selector.py`

**Command:**
```
python layout_selector.py 1
```

**First ever input:** `assets/para/para_1.json`
```json
{
  "para_1": "Your full paragraph of text goes here. It can be multiple sentences..."
}
```

**What it does (2 AI calls via OpenRouter):**
1. **Scene breaker** — splits the paragraph into scenes (max 320 chars each, exact consecutive sentences, no paraphrasing)
2. **Layout assigner** — AI picks top-3 ranked layouts per scene, applies last-5 no-repeat rule

**Output:** one file per scene in `assets/scene_prompts/`
```
assets/scene_prompts/scene_1_prompt.json
assets/scene_prompts/scene_2_prompt.json
...
```

Each file looks like:
```json
{
    "prompt": "scene_3: The exact sentence text from the paragraph.",
    "layout": "8",
    "para_id": "1"
}
```

## PHASE 2 — Director Script Generation

**Input:** `assets/scene_prompts/scene_N_prompt.json`

### Option A — Web UI (preview + edit)

```
python layout_tester.py
```
Open `http://localhost:5555` — select layout + scene number, click **Run Pipeline**.
Streams live logs. After completion, you can preview, edit, and adjust on the canvas.

### Option B — Automated batch

```
python run_and_audit.py [start_scene]
```
Processes all scenes from `start_scene` onward. Automatically audits each director script for empty phrases and cross-group duplicates.

### What both options do per scene:

For a scene assigned to layout `8`:
1. Run `layout_8_step_1.py N` — AI step: parse narrative, extract elements
2. Run `layout_8_step_2.py N` — AI step: add visual detail, image descriptions
3. Run `layout_8_generator.py N` — deterministic Python math: place elements on 1920x1080 canvas

Intermediate AI outputs go to `assets/tmp/`.

**Output:** `assets/directorscript/scene_N_director.json`
```json
{
  "scene_id": "scene_3",
  "elements": [
    {
      "element_id": "img_1",
      "type": "image",
      "x": 640,
      "y": 540,
      "scale": 0.4,
      "angle": 0,
      "phrase": "industrial robot",
      "description": "A realistic photo of an industrial robot arm",
      "filename": "robot.png",
      "animation": "pop",
      "property": "shadow"
    },
    {
      "element_id": "text_1",
      "type": "text_black",
      "x": 1400,
      "y": 400,
      "scale": 1.2,
      "angle": 0,
      "phrase": "industrial robot",
      "description": "",
      "filename": "",
      "animation": "slide_in_right",
      "property": "none"
    }
  ]
}
```

**Element types:** `image`, `arrow`, `text_black`, `text_red`, `text_highlighted`
**Animations:** `pop`, `slide_in_left`, `slide_in_right`, `slide_in_up`, `typing`, `fade_up`, `none`
**Properties:** `shadow`, `none`

## PHASE 3 — Image Generation

### Step 3a — Generate image prompts

**Script:** `generate_prompts.py`

**Command:**
```
python generate_prompts.py 3
```

**Input:** `assets/directorscript/scene_3_director.json`

Reads all image elements, calls AI to convert each element `description` field into a detailed image generation prompt.

**Output:** `assets/image_prompts/scene_3_image_prompts.json`

---

### Step 3b — Generate images on RunPod

**Script:** `image.py`

**Command:**
```
python image.py
```

**Input:** first JSON file found in `assets/image_prompts/`

Sends each element's image prompt to **ComfyUI running on RunPod** (remote GPU). Uses the Lumina2 model. Downloads each generated image via WebSocket when done.

**Output:** `assets/outputs/scene_N/filename.png` per element

> **Note:** `assets/outputs/arrow.png` is a global shared asset used by all layouts.

---

## PHASE 4 — Timings Generation

**Script:** `assets/timings.py`

**Command:**
```
cd assets
python timings.py a         # all scenes
python timings.py 1 2 3     # specific scenes
```

**Inputs:**
- `directorscript/scene_N_director.json` — element IDs and their `phrase` fields
- `scenes_audio/scene_N_transcript_full.json` — word-level timestamps from Whisper

**What it does:**
- Matches each element's `phrase` string against the spoken word transcript
- Finds the exact second that phrase is spoken in the audio
- Elements that appear together share a phrase and get the same start time

**Output:** `timings/scene_N_timings.json`
```json
[
  { "element_id": "img_1", "filename": "robot.png", "phrase": "industrial robot", "start": 2.4, "end": 8.1 },
  { "element_id": "text_1", "phrase": "industrial robot", "start": 2.4, "end": 8.1 }
]
```

---

## PHASE 5 — Final Video Render

**Script:** `render_video.py`

**Command:**
```
python render_video.py
```
Open `http://localhost:5566` — select scene number, quality, hardware, then click **Render**.

**Inputs:**
- `assets/directorscript/scene_N_director.json`
- `assets/timings/scene_N_timings.json`
- `assets/scenes_audio/scene_N.mp3`
- `assets/outputs/scene_N/*.png` (generated images)

**What it does:**
1. Mixes SFX sounds onto the voiceover at the correct timestamps (pop, whoosh, click, typing sounds matched to animation type)
2. Builds a white 1920x1080 (or 4K 3840x2160) canvas
3. Composites each element clip with its animation (pop, slide, fade, typing reveal)
4. Exports final video

**Final output:** `scene_N.mp4` in the project root

## Layout Reference

| # | Name | Best For |
|---|------|----------|
| 1 | Four Corners | 4 distinct items of equal importance |
| 2 | Diagonal Split | Before/after, two contrasting concepts |
| 3 | Horizontal Banners | Single punchy statement + one visual |
| 4 | Z-Pattern Flow | Cause-and-effect or step-by-step chains |
| 5 | Cause to Effect | Simple causal statements: X leads to Y |
| 6 | Two Rows | Two related but distinct concepts |
| 7 | Host Center + Sides | Narrator-driven scenes |
| 8 | Timeline Path | Processes, sequences, timelines |
| 9 | Comic Grid | Exactly 4 items or categories |
| 10 | Macro and Micro | One big idea + 3 supporting details |
| 11 | Mind-Map Orbit | A topic fanning out into sub-ideas |
| 12 | VS Comparison | Direct comparison between two things |
| 13 | Staircase | Progressive steps that accumulate |
| 14 | Circular Flow | Ongoing loops or recurring cycles |
| 15 | Hero + Bullets | One key claim + 2-4 supporting facts |
| 16 | Filmstrip | Sequential frames, nostalgia, memories |
| 17 | Triangle (Triforce) | Exactly 3 interconnected pillars |
| 18 | Polaroid Wall | Memory snapshots, nostalgic examples |
